# 📖 Project Overview

The **Titanic Exploratory Data Analysis (EDA)** project focuses on understanding the famous Titanic dataset through statistical analysis and data visualization.

The primary objective is to clean the dataset, explore passenger information, identify survival patterns, detect correlations, and generate meaningful insights using Python.

This project demonstrates essential **Data Analysis**, **Data Cleaning**, and **Visualization** skills that form the foundation of every Data Science workflow.

---

# 🎯 Objectives

- Explore the Titanic dataset
- Understand passenger demographics
- Handle missing and duplicate data
- Perform statistical analysis
- Visualize important features
- Identify survival trends
- Discover relationships among numerical variables
- Generate meaningful business insights

# 🔍 Exploratory Data Analysis Performed

### 📌 Data Inspection

- Dataset Shape
- Column Information
- Data Types
- First & Last Records
- Statistical Summary

### 🧹 Data Cleaning

- Removed Duplicate Records
- Filled Missing Age Values using Median

# 📈 Key Insights

Some important observations obtained during the analysis include:

- Female passengers had a higher survival rate than male passengers.
- First-class passengers showed better survival chances compared to lower classes.
- Missing values were successfully handled to improve data quality.
- Passenger age follows a moderately right-skewed distribution.
- Fare varies significantly across passenger classes.
- The boxplot helped identify potential age outliers.
- Correlation analysis revealed relationships among numerical features.
- Pairplot provided a clear overview of feature interactions.


